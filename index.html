PGuard

PGuard is a rugged security robot for multi-terrain applications.

PGuard is fully autonomous and self-driving gathering significant amounts of data from the environment in real-time and deters negative behavior.

PGuard provides security professionals with a force multiplier through an interface to review events generated from smart “eyes and ears”.

Overview

Automatic license plate recognition (ALPR) is considered among the cornerstones of video analytics in smart cities, particularly in security-related applications.

Vast range of applications benefit from this technology such as police surveillance, parking man agement, access control, tolling, and ITS.

To provide a valuable resource for the ALPR research community, we introduce PGuard ALPR Datasets specifically curated to address the in-the-wild ALPR scenario.

ELP Dataset

A comprehensive, geographically diverse image dataset for European license plates.

Key features

  • 4744 images, 5000 license plates.
  • 20 European countries, 10 vehicle categories.
  • Annotations: bounding box coordinates, four vertices locations, license plate number.

PGTLP Dataset

A diverse image dataset for Tunisian license plates.

Key features

  • 5000 HD images, 7244 license plates.
  • Multiple license plate per image.
  • Annotations: bounding box coordinates, four vertices locations, license plate number.
  • Areas: parkings, gates, roundabouts, roadsides.

PGFLP Dataset

A large and diverse image dataset for French license plates.

Key features

  • 9011 HD images, 14441 license plates.
  • Multiple license plate per image.
  • Annotations: bounding box coordinates, four vertices locations, license plate number.
  • Areas: parkings, roadsides.
